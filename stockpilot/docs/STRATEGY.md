## 1. 전략 철학

**추세추종 (Trend Following)** 기반.

"오르는 건 더 오른다" — 이미 방향이 정해진 종목을 따라가되,
눌림목(일시 조정) 구간에서 진입해 리스크/리워드를 최적화한다.

역추세(바이더딥) 전략은 채택하지 않는다.
하락 추세 중 저점 매수는 손실이 눈덩이처럼 커지는 구조이기 때문.

## 2. 매수 진입 조건 (B+C 조합)

세 가지 조건을 **모두** 충족해야 진입한다.

| 조건 | 기준 | 의미 |
|------|------|------|
| ① 주봉 추세 | 주봉 SMA5 > SMA10 | 중기 상승 추세 확인 |
| ② SMA20 지지 | 현재가 > 일봉 SMA20 | 단기 추세 유지 (C형) |
| ③ RSI 범위 | 일봉 RSI 40~60 | 눌림목 진입 타이밍 (B형) |

### 조건별 근거

**① 주봉 추세** — 큰 방향을 먼저 확인한다. 주봉이 하락 중이면
일봉 매수 시그널이 나와도 진입하지 않는다.

**② SMA20 지지 (C형)** — RSI 숫자보다 추세선 위에 있는지가 더 중요하다.
SMA20 아래로 내려온 종목은 추세가 이탈된 것으로 보고 진입을 금지한다.

**③ RSI 40~60 (B형)** — RSI 40 미만은 하락 압력이 강한 구간,
60 초과는 이미 달아오른 구간이다. 40~60은 상승 추세 중 숨 고르기 구간으로
진입 타이밍으로 가장 적합하다.

## 3. 매도 조건 (우선순위 순)

### ① 하드 스탑 (최우선)
```
평단 대비 -3% 이탈 → 무조건 손절
```
- 감정 개입 없이 기계적으로 실행
- 다른 조건보다 항상 우선
- 최대 손실을 -3%로 고정

### ② 트레일링 스탑
```
평단 +2% 돌파 시 활성화
→ 이후 5일 고가 기준 -3% 이탈 시 청산
```
- +2% 돌파 순간부터 최소 수익 청산 보장
- 추세가 강할 때 +5% 이상도 먹을 수 있음
- 5일 고가를 기준으로 잡아 자동화 용이

### ③ 목표가 익절
```
평단 대비 +5% 도달 → 익절
```
- 트레일링 활성화 전 +5% 직행 시 적용
- 트레일링과 동시에 체크, 먼저 도달하는 조건 적용

### ④ 보류 청산 (기회비용 방지)
```
5일 경과 + 5일 고가가 평단 +2% 미달 → 청산
```
- 진입 후 횡보하며 기회비용이 발생하는 상황 방지
- +2% 트레일링 활성화가 한 번도 안 됐을 때만 적용

## 4. 리스크/리워드

```
최대 손실:    -3.0% (하드 스탑 — 고정)
기본 익절:    +5.0%
트레일링:     +2.0% ~ 무제한 (추세 강도에 따라)

기본 R:R     = 1 : 1.67
트레일링 R:R = 1 : 무제한
```

## 5. 시나리오별 결과

| 시나리오 | 흐름 | 결과 |
|---------|------|------|
| 손절 | 진입 후 바로 하락 → 평단 -3% | **-3.0%** |

| 트레일링 — 소폭 | +2% 돌파 후 고가 -3% 이탈 | **+0% ~ +2%** |

| 보류 청산 | 5일 횡보 후 고가 +1% | **+1% 근처 청산** |

## 6. 현재 보유 종목에 대한 전략 적용 기준

### 신규 진입 기준
- 위 3가지 진입 조건 모두 충족 시에만 매수
- RSI 60 초과 구간에서는 추가매수 금지

### 기존 보유 종목 물타기 기준
| 조건 | 판단 |
|------|------|
| RSI < 40 + SMA20 위 | 🟢 분할 추가매수 검토 가능 |
| RSI 40~60 + SMA20 위 | 🟡 홀드 유지 |
| RSI > 60 | 🟡 추가매수 보류 |
| SMA20 하회 | 🔴 추가매수 금지 |
| 평단 -3% 이탈 | 🚨 즉시 손절 |

## 7. 설정 파일

수치 조정은 `data/strategy_config.json`에서만 한다.
코드를 수정하지 않아도 전략 수치 변경이 전체에 반영된다.

```json
{
  "entry": {
    "rsi_range": { "min": 40, "max": 60 }
  },
  "exit": {
    "hard_stop":     { "pct": -3.0 },
    "trailing_stop": { "activation_pct": 2.0, "trail_pct": 3.0, "lookback_days": 5 },
    "take_profit":   { "pct": 5.0 },
    "stale_exit":    { "days": 5, "min_gain_pct": 2.0 }
  }
}
```
