## 1. 시스템 개요

한국투자증권(KIS) Open API를 활용해 매일 자동으로 주식 브리핑을 카카오톡으로 전송하는 시스템입니다.

### 1.1 자동 실행 일정 (평일 기준)

| 시각 | 스크립트 | 내용 |
|------|----------|------|
| 08:20 | `watchlist_sync.py` | KIS HTS 관심종목 → watchlist.json 동기화 |
| 08:30 | `morning_report.py` | 모닝 브리핑 카카오톡 전송 |
| 09:10 | `intraday_report.py` | 장초기 현황 카카오톡 전송 |
| 20:30 | `closing_report.py` | 장마감 결산 카카오톡 전송 (넥스트장 포함) |
| 23:30 | `stock_discovery.py` | 야간 종목 발굴 카카오톡 전송 |

### 1.2 프로젝트 디렉토리 구조

```
stockpilot/
├── morning_report/          # 핵심 스크립트
│   ├── morning_report.py    # 모닝 브리핑
│   ├── intraday_report.py   # 장초기 브리핑
│   ├── closing_report.py    # 장마감 결산
│   ├── stock_discovery.py   # 야간 종목 발굴
│   ├── watchlist_sync.py    # 관심종목 동기화
│   ├── kakao_sender.py      # 카카오톡 전송 모듈
│   ├── keychain_manager.py  # macOS Keychain 인증정보 관리
│   └── setup_scheduler.sh   # launchd 스케줄러 등록 스크립트
├── .skills/
│   └── kis-api/scripts/
│       └── kis_client.py    # KIS API 클라이언트
├── data/
│   ├── watchlist.json       # 관심종목 목록
│   ├── cache/               # 토큰 캐시 (KIS, 카카오)
│   └── raw/                 # 일봉 데이터 캐시
├── docs/
│   ├── api/                 # KIS 전체 API 문서 (xlsx)
│   └── manual/              # 이 매뉴얼
├── logs/                    # 자동 실행 로그
├── reports/

├── venv/                    # Python 가상환경
└── .env                     # 환경변수 (민감정보 제외)
```

## 2. 초기 설치

### 2.1 사전 요구사항
- macOS (Keychain 사용)
- Python 3.11 이상
- 한국투자증권 계좌 + KIS Developers 앱 등록

### 2.2 가상환경 설치

```bash
cd /Users/geenya/projects/AI_Projects/stockpilot
python3 -m venv venv
venv/bin/pip install -r morning_report/requirements.txt
venv/bin/pip install keyring
```

### 2.3 인증정보 등록 (Keychain)

```bash
venv/bin/python3 morning_report/keychain_manager.py --reset
```

입력 항목:
- **KIS_APP_KEY**: KIS 개발자센터 → 내 앱 → 앱키
- **KIS_APP_SECRET**: KIS 개발자센터 → 내 앱 → 앱시크릿
- **KIS_ACCOUNT_NO**: 계좌번호 (형식: 12345678-01)
- **KIS_HTS_ID**: MTS/홈페이지 로그인 ID

입력 후 자동으로 KIS API 연결 테스트 진행 → 성공 시 Keychain 저장.

### 2.4 스케줄러 등록

```bash
./morning_report/setup_scheduler.sh
```

## 3. 인증정보 관리

### 3.1 Keychain 저장 구조

| 항목 | Keychain 키 | 설명 |
|------|-------------|------|
| KIS 앱키 | `KIS_APP_KEY` | KIS 개발자센터 발급 |
| KIS 앱시크릿 | `KIS_APP_SECRET` | KIS 개발자센터 발급 |
| 계좌번호 | `KIS_ACCOUNT_NO` | 12345678-01 형식 |
| 로그인 ID | `KIS_HTS_ID` | MTS 로그인 ID |

서비스명: `AI주식매매` (macOS 키체인 앱에서 검색 가능)

### 3.2 인증정보 변경 방법

```bash
# 전체 재입력
venv/bin/python3 morning_report/keychain_manager.py --reset

# 특정 항목만 변경
venv/bin/python3 morning_report/keychain_manager.py --reset KIS_APP_KEY
venv/bin/python3 morning_report/keychain_manager.py --reset KIS_HTS_ID

# 현재 저장 상태 확인
venv/bin/python3 morning_report/keychain_manager.py
```

### 3.3 카카오 토큰

카카오 관련 키는 `.env` 파일에 저장:
- `KAKAO_REST_API_KEY`: 카카오 앱 REST API 키
- `KAKAO_REFRESH_TOKEN`: 자동 갱신됨 (만료 시 `setup_kakao.py` 재실행)
- `KAKAO_CLIENT_SECRET`: 카카오 앱 시크릿

## 4. 스케줄러 관리

### 4.1 등록된 launchd plist 목록

| plist 파일 | 실행 시각 | 스크립트 |
|------------|-----------|----------|
| `com.aigeenya.stockreport.watchlist.plist` | 평일 08:20 | watchlist_sync.py |
| `com.aigeenya.stockreport.plist` | 평일 08:30 | morning_report.py |
| `com.aigeenya.stockreport.intraday.plist` | 평일 09:10 | intraday_report.py |
| `com.aigeenya.stockreport.closing.plist` | 평일 20:30 | closing_report.py |
| `com.aigeenya.stockreport.discovery.plist` | 월~토 23:30 | stock_discovery.py |

### 4.2 스케줄 확인 및 관리

```bash
# 등록 확인
launchctl list | grep stockreport

# 전체 재등록 (시각 변경 시)
./morning_report/setup_scheduler.sh

# 특정 스케줄 중지
launchctl unload ~/Library/LaunchAgents/com.aigeenya.stockreport.closing.plist

# 특정 스케줄 재시작
launchctl load ~/Library/LaunchAgents/com.aigeenya.stockreport.closing.plist
```

### 4.3 주의사항
- Mac이 해당 시각에 켜져(또는 깨어) 있어야 함
- 공휴일은 launchd가 인식 불가 → KIS API 빈 데이터로 자동 감지 후 조용히 종료
- 배터리 설정 → "전원 어댑터 사용 시 잠자지 않게 하기" ON 권장

## 5. 관심종목(watchlist) 관리

### 5.1 자동 동기화 방식

매일 08:20에 `watchlist_sync.py`가 실행되어:
1. KIS HTS 관심종목 그룹 조회 (TR: HHKCM113004C7)
2. 그룹별 종목 조회 (TR: HHKCM113004C6)
3. 현재 보유 종목 자동 추가
4. `data/watchlist.json` 업데이트

### 5.2 수동 실행 명령어

```bash
# 변경 내용 미리 확인 (저장 안 함)
venv/bin/python3 morning_report/watchlist_sync.py --dry-run

# 실제 동기화 실행
venv/bin/python3 morning_report/watchlist_sync.py

# 현재 watchlist 내용 확인
venv/bin/python3 morning_report/watchlist_sync.py --show
```

### 5.3 섹터 매핑

- KIS HTS 그룹명이 섹터로 자동 지정됨
- 기존 섹터값 우선 유지 (변경 안 됨)
- 신규 종목은 그룹명 또는 "기타"로 추가
- `watchlist_sync.py` 내 `_SECTOR_MAP`에 수동 추가 가능

## 7. 장애 대응

### 7.1 카카오톡 전송 실패

1. 토큰 만료 여부 확인:
   ```bash
   venv/bin/python3 morning_report/kakao_sender.py
   ```
2. 실패 시 `setup_kakao.py` 재실행하여 토큰 재발급
3. 전송 실패한 보고서는 `reports/` 폴더에 `.txt`로 저장됨

### 7.2 KIS API 오류

1. Keychain 인증정보 확인:
   ```bash
   venv/bin/python3 morning_report/keychain_manager.py
   ```
2. 토큰 캐시 초기화:
   ```bash
   rm data/cache/kis_token.json
   ```
3. API 연결 재테스트:
   ```bash
   venv/bin/python3 morning_report/keychain_manager.py --reset
   ```

### 7.3 스크립트 수동 테스트

```bash
# dry-run (카카오 전송 없이 출력만)
venv/bin/python3 morning_report/morning_report.py --dry-run
venv/bin/python3 morning_report/intraday_report.py --dry-run
venv/bin/python3 morning_report/closing_report.py --dry-run
venv/bin/python3 morning_report/stock_discovery.py --dry-run
```
