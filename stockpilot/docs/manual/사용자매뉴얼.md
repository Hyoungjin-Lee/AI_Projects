## 3. 카카오톡 메시지 읽는 법

### 3.1 모닝 브리핑 (08:30)

```
📊 2026년 04월 17일 (금) 모닝 브리핑
============================
📈 오늘 시장 전망
  ...

💼 보유종목 전략
🔴 GS건설(006360)
  현재가: 39,000원  |  평가손익: -205,350원 (-4.17%)
  📌 오늘 전략: 손절가 33,313원 유지, 반등 시 관망
```

- 🟢 = 수익 중 / 🔴 = 손실 중
- 📌 = 오늘 대응 전략

### 3.2 장마감 결산 (20:30)

```
📊 2026년 04월 17일 (금) 장마감 결산
⏰ 20:30 기준
============================
🔴 GS건설(006360)
  종가: 39,000원  ━0.00%  |  평가손익: -205,350원 (-4.17%)
  고가: 39,500  /  저가: 38,200  |  거래량: 2,015,463주
  📊 도지 (방향성 불명확)
  💤 거래량 급감 (평균 대비 0.4배) — 관망세
  📌 내일 전략: 손절가: 33,313원 | 목표가: 50,374원
============================
📋 오늘 결산 요약
  주식 평가액:     8,443,500원
  예수금(현금):    4,751,541원
  총자산:         13,195,041원
  🔴 오늘 평가손익: -250,350원 (투자금 대비 -2.88%)
```

**거래량 신호 읽는 법:**
- 🚨 거래량 급증 (2.5배↑) = 세력 유입 또는 재료 발생 주의
- ⚡ 거래량 증가 (1.5배↑) = 관심 증가
- 💤 거래량 급감 (0.5배↓) = 관망세

**캔들 신호 읽는 법:**
- 강한 양봉 = 상승 모멘텀 → 홀드 or 추가 매수 검토
- 강한 음봉 = 하락 압력 → 손절가 재확인
- 도지 = 방향성 불명확 → 관망
- 긴 윗꼬리 = 고점 매도 압력 → 추격 매수 주의
- 긴 아랫꼬리 = 저점 매수세 유입 → 반등 가능성

**MTS에서 관심종목 추가하는 법:**
1. 종목 검색
2. 종목 상세 화면 → ☆(즐겨찾기) 터치

## 5. 매매일지 확인

장마감 결산이 실행될 때마다 매매일지가 자동 저장됩니다.

**저장 위치:**
```
reports/journal/journal_YYYYMMDD.md
```

**예시:**
```
reports/journal/journal_20260417.md
```

Finder에서 열거나 메모앱, 노션 등에 복사해서 활용하세요.  
각 종목 아래 **📝 메모 (직접 작성)** 섹션에 오늘 매매 이유, 특이사항 등을 기록해두면 나중에 복기할 때 유용합니다.

## 7. 주의사항

1. **실주문 기능은 현재 비활성화** 상태입니다. 브리핑과 분석만 제공하며 실제 매수/매도는 직접 하셔야 합니다.
