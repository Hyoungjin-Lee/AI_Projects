# 📘 stockpilot 전략 — 중학생도 이해하는 완전 가이드

> 작성일: 2026-04-28
> 대상: 비전공자, 처음 보는 사람도 이해할 수 있게 비유로 설명
> 범위: 현재 운영 중인 전략 + 새로 검토 중인 패턴 매매 + 향후 적용 아이디어

## 0. 이 프로그램이 뭐 하는 거야?

**stockpilot** = 한국 주식을 자동으로 사고팔게 도와주는 AI 비서.

비유로 말하면:
> "체육시간에 어느 팀에 들어갈지 고민될 때, 옆에서 '쟤네 팀이 이기고 있으니까 거기 들어가'라고 알려주는 친구."

핵심: **이미 잘 나가는(오르고 있는) 종목**만 따라가서 산다. 떨어지고 있는 종목은 안 산다.

## 1. 우리가 쓰고 있는 기본 전략 (B+C 조합)

### 1-1. 큰 원칙: "추세추종" — 흐름 따라가기

**비유**: 운동장에서 친구들이 한쪽으로 우르르 뛰고 있어. 그 방향이 맞을 가능성이 높지? 그러면 같은 방향으로 따라 뛰는 게 "추세추종"이야.

반대로, "다들 저쪽으로 가는데 나는 반대로 가야지" = 역추세. 멋있어 보이지만 대부분 다친다. **우리는 반대로 안 간다.**

### 1-2. 살까 말까 결정하는 3가지 조건

이 3개가 **모두** 맞아야 산다. 하나라도 빠지면 안 산다.

#### ① 주봉 추세 (큰 그림)
- "주봉" = 일주일 단위로 묶은 차트
- 조건: 주봉 SMA5 > SMA10
- 쉬운 말: **"최근 5주 평균 가격이 최근 10주 평균보다 높다"**
- 의미: 큰 흐름이 우상향이라는 뜻. 멀리서 봐도 오르고 있어야 함

> 비유: 한 달 전보다 1주일 전이 더 좋아진 학생. "성적이 오름세" 같은 거.

#### ② SMA20 위에 있어야 함 (단기 흐름 유지)
- "SMA20" = 최근 20일 평균 가격선
- 조건: 지금 주가 > 일봉 SMA20
- 의미: 단기적으로도 평균선 위에 있어야 함. 평균선 아래로 떨어지면 추세가 꺾인 것

> 비유: 평소 평균 점수 70점인 학생이 오늘 시험 65점이면 **떨어진 것**. 75점이면 평균 위에 있는 것. 우리는 **평균 위인 학생**만 응원함.

#### ③ RSI 40~60 사이 (숨 고르기 구간)
- "RSI" = 0~100 사이의 숫자. 주가가 얼마나 뜨거운지/식었는지 보여줌
  - RSI 0~30: 너무 차가움 (과매도, 떨어질 만큼 떨어짐)
  - RSI 30~40: 약간 식음
  - **RSI 40~60: 적당함 (여기서 사야 함!)**
  - RSI 60~70: 약간 뜨거움
  - RSI 70~100: 너무 뜨거움 (과매수, 곧 떨어질 수 있음)
- 의미: 너무 과열되지도, 너무 식지도 않은 **적당한 시점**

> 비유: 100m 달리기 직후 헐떡이는 친구(RSI 80)랑 잠 자는 친구(RSI 20) 말고, **숨 고르고 막 다시 뛰려는 친구(RSI 50)**가 가장 좋은 컨디션.

### 1-3. 팔 때는 4가지 규칙 (우선순위 순)

산 다음에는 언제 팔까? 4가지 규칙을 순서대로 체크.

#### ① 손절 (-3%) — 가장 중요!
```
산 가격보다 -3% 떨어지면 → 무조건 판다
```
> 비유: 게임에서 체력이 30% 이하면 무조건 도망치는 룰. 감정 빼고 기계처럼.

**왜?** 한 번에 큰 손실 막기 위해. -3% 손해는 다음에 +3.1% 벌면 회복되지만, -50% 손해는 +100% 벌어야 본전.

#### ② 트레일링 스탑 (수익 따라가기)
```
+2% 넘으면 활성화 → 그 후 5일 최고가에서 -3% 떨어지면 판다
```
> 비유: 시험 점수가 80점에서 90점, 95점, 92점, 88점 이렇게 떨어지면 → **최고점 95점 대비 7점 빠진 88점에서 멈춤**. 더 떨어지기 전에 챙김.

**왜?** 잘 나가는 동안은 끝까지 같이 가되, 꺾이면 바로 챙겨 나옴.

#### ③ 목표가 익절 (+5%)
```
산 가격보다 +5% 오르면 → 판다
```
> 비유: 게임에서 점수 1000점 도달하면 일단 저장하고 끝내는 것.

#### ④ 보류 청산 (5일 횡보)
```
샀는데 5일 동안 +2%도 못 오르면 → 판다
```
> 비유: 5일 동안 게임에서 한 판도 못 이긴 캐릭터는 다른 캐릭터로 바꾸는 것. 시간 낭비 방지.

### 1-4. 한눈에 보기

**살 때**: 주봉 우상향 + 평균선 위 + RSI 적당  
**팔 때**: -3% 손절 / +5% 익절 / 트레일링 / 5일 횡보

## 2. 새로 검토 중인 패턴 매매법 (PPT 자료 분석)

사용자가 받은 자료에는 **5가지 매매법**이 있어. 이건 비트코인용으로 만들어진 거지만, 우리 stockpilot에 일부 가져올 수 있을지 검토 중.

### 2-1. 5가지 매매법 한눈에

| # | 이름 | 한 줄 설명 | 방향 |
|---|---|---|---|
| 1 | **바닥** | 떨어지다가 바닥 다지고 다시 올라가는 모양 | 사기 (롱) |
| 2 | **뚜껑** | 오르다가 천장 막혀서 다시 내려가는 모양 | 팔기 (숏) |
| 3 | **서치** | 가짜로 받쳐주다가 갑자기 또 떨어지는 함정 | 팔기 (숏) |
| 4 | **공급** | 평소 잘 받쳐주는 선에서 매수가 들어와 올라감 | 사기 (롱) |
| 5 | **라인** | 오르는 추세선을 따라 계속 매수가 들어옴 | 사기 (롱) |

### 2-2. 각 패턴 자세히

#### ① 바닥 패턴 — "여기가 바닥인가?"
**모양**: 떨어지던 가격이 어떤 지점에서 멈추고 매수자들이 들어와 **긴 아래꼬리**(↓⬇↑)를 만들면서 바닥을 다짐.

> 비유: 공이 바닥에 떨어졌다가 통! 하고 튕겨 오르는 모양. 통 튕기는 그 순간이 "바닥".

**판단 기준**:
- 거래량이 들어왔다 빠졌다 반복 (사람들이 사고 팔고 사고 팔고)
- 직전 저점을 잠깐 깨고 다시 올라옴 = 마지막 손절 매물 털어내기
- 그 순간 매수 진입

**stockpilot 적용?** △ 보유 중인 종목 추가 매수 신호로만 활용. 새로 진입은 안 함 (역추세 성격이라).

#### ② 뚜껑 패턴 — "여기가 천장인가?"
**모양**: 오르던 가격이 어느 지점에서 막히면서 거래량은 줄고 가격만 위로 살짝 오름. **거래량 다이버전스(diverge=어긋남)**.

> 비유: 점프 잘하던 친구가 점점 힘이 빠져서 같은 자리에서만 폴짝거리는 모양. 곧 주저앉음.

**판단 기준**:
- 1차 뚜껑(거래량 많은 상승봉) → 2~3봉 동안 거래량 줄면서 가격만 살짝 오름 → 2차 뚜껑 → 3차까지 확인되면 하락 시작

**stockpilot 적용?** △ 보유 중인 종목 **매도 신호**로만 활용. 우리는 숏(공매도)를 안 하니까.

#### ③ 서치 패턴 — "이거 함정이야!" ⚠️
**모양 (사용자 설명 그대로)**:
> "고점 올라갔는데 막 내려가. 근데 어~ 애들이 물량을 받혔어. 그럼 다시 올라가겠지? 근데 어느 지점에서 내리 꼽는 구간."

> 비유: 게임에서 적이 일부러 약한 척 도망가다가 뒤에서 매복으로 패는 거. 따라가면 죽음.

**판단 기준**:
1. 가격이 고점에서 빠르게 떨어짐
2. 매수자들이 "어, 싸졌네!" 하고 들어옴 (거래량 발생)
3. 잠깐 반등할 것 같음 → **함정**
4. 갑자기 또 떨어짐 → 들어온 매수자들 손절 → 더 떨어짐

**stockpilot 적용?** ◎ **매우 유용함!**
- 우리는 숏 안 하지만, "이런 패턴 보이면 **사지 마라**" 하는 **방어 신호**로 쓸 수 있음
- 진입 거부권: B+C 조건 다 맞아도 서치 패턴 보이면 매수 보류
- 손절 가속기: 보유 중인데 서치 패턴 나오면 -3% 기다리지 말고 즉시 매도

#### ④ 공급 패턴 — "받쳐주는 손이 있다"
**모양**: 차트에 그어둔 가로선(예전부터 그 가격에서 잘 받쳐주던 선)에서 매수세가 들어와 다시 올라감.

> 비유: 공이 트램폴린 위로 떨어지면 통 하고 튀어오름. 트램폴린 = 지지선.

**stockpilot 적용?** ◯ 우리 전략(추세추종)이랑 잘 맞음. 신규 매수 보조 신호로 활용 가능.

#### ⑤ 라인 패턴 — "길잡이 선을 따라가"
**모양**: 가격이 오를 때 일정한 선(추세선)을 따라 계속 매수가 들어와 올라감.

> 비유: 에스컬레이터 손잡이를 잡고 천천히 올라가는 모양.

**stockpilot 적용?** ◯ 우리의 SMA20 지지 조건이랑 본질이 같음. 더 정밀하게 만들 수 있음.

### 2-3. stockpilot에 적용 우선순위

| 순위 | 패턴 | 이유 |
|---|---|---|
| 🥇 1순위 | **서치 (방어용)** | 즉시 효과, 만들기 쉬움. "안 사야 할 때 안 사는" 능력이 손익에 가장 큼 |
| 🥈 2순위 | **공급 / 라인** | 우리 추세추종 전략이랑 정합성 좋음 |
| 🥉 3순위 | **바닥 / 뚜껑** | 정책 결정 후. 철학 충돌 가능성 있음 |

## 3. 성공률을 높이기 위한 13가지 추가 아이디어

### A 그룹: 신호를 더 정확하게

#### A1. 다중 시간대 합의 (Confidence Cascade)
> 비유: 한 친구만 "쟤가 좋다"고 하면 안 믿지. 3명이 다 "쟤 좋다"고 하면 믿음 증가.

- 1시간 차트만 신호 → 약함
- 1시간 + 15분 → 보통
- 1시간 + 15분 + 5분 → 강함

#### A2. 지표 + 패턴 점수 합산 (Convergence Score)
> 비유: 학원 시험(B+C 지표)이랑 학교 시험(패턴) 둘 다 잘 보면 진짜 잘하는 거.

- 둘 다 매수 신호 = 진입
- 한쪽만 신호 = 보류
- **반대로 신호 = 절대 금지**

#### A3. 호가창 행동 확인 (Order Flow)
> 비유: 식당에 사람들이 줄 서 있는지(호가) 보면 진짜 인기 식당인지 알 수 있음.

- 패턴 + 매수 호가 누적 = 진짜 신호
- 패턴 + 매수 호가 비어있음 = 가짜일 가능성

### B 그룹: 한국 시장 특화

#### B1. 외국인/기관 흐름 확인
> 비유: 인기 학생이 누구랑 친한지 보면 진짜 인기인지 가짜 인기인지 알 수 있음.

- 패턴 신호 + 외국인/기관 매수 = 신뢰도 ↑
- 패턴 신호 + 개인만 매수 = 의심

#### B2. 시간대별 가중치
> 비유: 같은 농구 슛도 경기 첫 1분, 점심 후 졸린 시간, 마지막 30초는 다름.

- 09:00~09:30: 시초가 변동 큼 → 신중
- 11:30~13:00: 점심시간 → 거래량 적음
- 14:30 이후: 마감 변동성 → 신중
- **10:00~11:30, 13:00~14:30**: 가장 신뢰할 만한 시간대

#### B3. 시장 분위기 필터
> 비유: 야외 축구 경기는 비 오는 날과 맑은 날 전략이 다름.

- 추세장(쭉쭉 오르거나 쭉쭉 내림) → 라인/공급 활성화
- 횡보장(오르락내리락) → 바닥/뚜껑 활성화
- **하락장 → 매수 전면 정지** (서치 필터만 가동)

### C 그룹: 돈 관리

#### C1. 신호 강도별 매수량 조절
> 비유: 70% 확신할 땐 용돈 절반, 90% 확신할 땐 용돈 전부.

- 약신호 = 평소의 절반
- 표준 신호 = 평소대로
- 강신호 = 평소의 1.5배

#### C2. 손실 누적 시 자동 보수화
> 비유: 게임에서 연속으로 죽으면 일단 안전한 곳에서 회복하는 것.

- 누적 -5% → 강신호만 거래
- 누적 -10% → **신규 매수 정지**, 기존 종목 정리만
- 누적 +10% → 정상 복귀

**왜 중요?** 사람은 연속 손실 후 "한 번에 만회하자" 생각해서 더 큰 실수를 함. 이걸 자동으로 막음.

#### C3. 실험용 자본 분리
> 비유: 용돈의 90%는 안전하게, 10%는 새로운 거 시도해보는 용도.

- 메인 자본 90% = 검증된 전략
- 실험 자본 10% = 새 패턴 테스트용
- 실험에서 검증되면 메인으로 승격

### D 그룹: 자동 학습

#### D1. 패턴 추적 통계
> 비유: 시험 끝날 때마다 "이번엔 이렇게 공부해서 몇 점 받았다" 기록하면 다음에 더 잘 공부할 수 있음.

- 모든 패턴 검출을 기록 → 24시간/72시간 후 결과 추적
- 100건 누적되면 "이 패턴 진짜 승률 X%" 자동 계산
- 그 결과로 임계값 자동 조정

#### D2. 실패 패턴 학습 (Anti-Pattern)
> 비유: 친구가 좋다고 추천한 영화 중 별로였던 것들 따로 기록해두면, 다음에 비슷한 추천 들어오면 안 봄.

- 검출했는데 실패한 패턴 따로 저장
- 새 신호가 과거 실패 패턴과 비슷하면 신뢰도 ↓

#### D3. AI 이중 검증 (Two-Person Rule)
> 비유: 중요한 결정은 친구 두 명한테 물어보고 결정하는 거.

- 1차: 알고리즘이 "사라" 신호
- 2차: AI가 차트 보고 "동의함"이라고 해야 진짜 매수
- 둘 중 하나라도 반대하면 안 삼

**stockpilot은 이미 AI 기반이라 자연스럽게 추가 가능.**

### E 그룹: 안전장치

#### E1. 단계적 도입 (Shadow → Alert → Trade)
> 비유: 새 게임 캐릭터 키울 때 처음엔 약한 적부터 잡는 것.

1. **Shadow (조용히)**: 검출만 하고 거래 X, 결과만 기록
2. **Alert (알림만)**: 텔레그램 알림 보내고 거래 X
3. **Trade-Small (소액)**: 1주씩만 사봄
4. **Trade-Full (정상)**: 정상 사이즈

각 단계에서 일정 기간 + 승률 기준 통과해야 다음 단계로.

#### E2. 긴급 정지 버튼 (Kill Switch)
> 비유: 게임에서 위험할 때 누르는 비상 정지 버튼.

조건:
- 일중 누적 손실 -3% 도달 → 패턴 매매 전면 정지
- 직전 20건 중 false positive 30% 초과 → 자동 비활성화
- 텔레그램 즉시 알림

```
┌─────────────────────────────────────────────────────────┐

  주봉 우상향? + SMA20 위? + RSI 40~60?
                          ↓
[3차 필터: 패턴 확인]
  바닥/공급/라인 패턴 검출? + 다중 시간대 합의?
                          ↓

  AI도 "사도 좋음"이라고 동의?
                          ↓

                          ↓
[매수 실행]
                          ↓
[보유 중 모니터링]
  -3% 손절 / +5% 익절 / 트레일링 / 5일 횡보 / 서치 감지
                          ↓
[매도 실행]
                          ↓
[기록 & 학습]
  성공/실패 기록 → 다음 임계값 자동 조정
```

## 5. 단계별 도입 계획

### Phase A — 인프라 (2주)
- 백테스트 환경 만들기 (과거 데이터로 시뮬레이션)
- 분봉/호가 데이터 끌어오기
- 라인 자동 검출 (지지선/저항선)

### Phase B — 서치 필터 도입 (2주)
- 서치 패턴 검출
- 매수 거부권 통합
- Shadow → Alert → Trade 단계적 적용

### Phase C — 공급/라인 도입 (3주)
- 공급/라인 패턴 검출
- B+C와 합쳐서 점수 계산
- 신호 강도별 매수량 차등

### Phase D — 확장 (지속)
- 바닥/뚜껑 (정책 결정 후)
- AI 이중 검증
- 외인/기관 게이트
- 자동 학습

## 6. 정리 — 가장 중요한 5가지

1. **추세 따라가기. 거꾸로 가지 말기.** (가장 중요한 철학)
2. **3가지 조건 다 맞아야 산다.** (주봉 + SMA20 + RSI)
3. **-3% 손절은 무조건. 감정 빼기.**
4. **"안 사는 능력"이 가장 큰 ROI.** (서치 필터)
5. **천천히, 단계적으로 도입.** (Shadow → Alert → Trade)
