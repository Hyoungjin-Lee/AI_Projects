# 리스크 관리 가이드

형진님 투자 시스템의 핵심 원칙. 어떤 전략을 쓰든 이 규칙이 우선한다.

---

## 1. 2% 룰 (포지션 최대 손실 제한)

**원칙**: 단일 포지션에서 발생할 수 있는 최대 손실은 **총 투자 자본의 2%**를 넘지 않는다.

**계산 방법**

```
포지션 수량 = (총 자본 × 0.02) / (진입가 - 손절가)
```

예시: 총 자본 5,000만 원, 삼성전자 진입가 70,000원, 손절가 68,000원

```
최대 손실 허용 = 5,000만 × 0.02 = 100만 원
손절 거리 = 70,000 - 68,000 = 2,000원
최대 수량 = 100만 / 2,000 = 500주
```

→ 500주 × 70,000 = 3,500만 원 포지션 (자본 대비 70%, 레버리지 주의)

**운용 팁**

- 단타는 1% 룰을 적용하는 것도 합리적 (높은 매매 빈도 고려)
- 동시에 5개 포지션을 보유하면 총 노출 리스크가 10%까지 늘어날 수 있음. 동시 보유 포지션 수를 제한할 것

## 2. ATR 기반 손절 라인

**원칙**: 임의의 금액이 아닌 **가격 변동성(ATR)에 비례한** 손절을 사용한다.

```
손절가 (매수) = 진입가 - ATR(14) × 배수
손절가 (매도) = 진입가 + ATR(14) × 배수
```

**배수 가이드라인**

| 전략 유형 | 권장 배수 | 이유 |
|---|---|---|
| 단타 (데이트레이딩) | 1.0~1.5× | 빠른 손절, 당일 정리 원칙 |
| 스윙 (며칠~몇 주) | 2.0~2.5× | 일봉 노이즈 허용 |
| 정량 (시스템) | 2.5~3.0× | 트렌드 추종, 여유 있게 |

`analyze_swing.py`는 ATR × 2를 기본 손절로 사용한다.

## 3. 켈리 베팅 (Kelly Criterion)

**개요**: 장기 자본 성장을 극대화하는 이론적 최적 포지션 크기. 실전에서는 **하프 켈리(½ Kelly)** 이하를 사용한다.

**켈리 공식**

```
f* = (p × b - q) / b

f* = 투자 비율
p  = 승률
q  = 패율 (1 - p)
b  = 평균 수익 / 평균 손실 (손익비)
```

예시: 승률 55%, 손익비 2:1

```
f* = (0.55 × 2 - 0.45) / 2 = (1.10 - 0.45) / 2 = 0.325
하프 켈리 = 0.325 × 0.5 = 16.25%
```

→ 자본의 최대 **16%**를 이 전략에 배분

**실전 주의사항**

- 켈리 값이 20% 초과 시 무시하고 10~15% 캡 적용
- 백테스트 승률·손익비를 과대추정하면 과도 배팅으로 파산 위험
- 슬리피지·세금을 포함한 순수 손익비로 계산할 것

## 4. 분할 매수 / 분할 매도

**분할 매수 원칙**

추세 확인 전 전액 진입하지 않는다. 3단계 분할을 권장한다.

| 단계 | 진입 조건 | 비중 |
|---|---|---|
| 1차 | 시그널 발생 (예: 골든크로스 확인) | 40% |
| 2차 | 1차 진입가 −2~3% 추가 조정 시 | 40% |
| 3차 | 추세 재확인 후 (신호 강도 높을 때) | 20% |

단타 전략은 분할 진입이 어려우므로 1회 진입 후 즉시 손절 원칙 적용.

**분할 매도 원칙**

목표가 도달 시 전량 매도보다 단계적 이익 실현을 권장한다.

| 단계 | 조건 | 비중 |
|---|---|---|
| 1차 | 목표가 50% 도달 시 | 30~40% |
| 2차 | 목표가 100% 도달 시 | 30~40% |
| 3차 | 트레일링 스탑 이탈 시 잔량 청산 | 나머지 |

## 5. 포트폴리오 분산 가이드

**종목 수**

- 단타: 동시 최대 3종목 (집중 모니터링 필요)
- 스윙: 동시 최대 5~7종목
- 정량: 동시 최대 10~15종목 (시스템이 관리하므로 가능)

**섹터 분산**

동일 섹터 비중이 포트폴리오의 30%를 넘지 않도록 한다. 반도체 3종목 + 반도체 ETF는 사실상 한 섹터 집중.

**자산 유형 분산**

| 유형 | 권장 비중 |
|---|---|
| 대형주 (코스피 시총 상위) | 50~60% |
| 중소형주·테마 | 20~30% |
| ETF/인버스 헤지 | 10~20% |

**현금 비중**

시장 불확실성이 높을 때(VIX 급등, 코스피 −5% 이상 급락 직후) 현금 비중을 30~50%로 높인다.

## 6. 연속 손절 프로토콜

**원칙**: 연속 3회 손절 발생 시 즉시 매매 중단, 1~2일 휴식 후 원인 분석.

**원인 분석 체크리스트**

1. 손절이 너무 촘촘한가? (ATR 배수 조정 필요)
2. 시장 국면이 전략에 맞지 않는가? (추세 vs 횡보)
3. 과신매매(오버트레이딩)인가?
4. 진입 규칙을 정확히 따랐는가? (심리적 임의 판단 배제)

**최대 일일 손실 한도**

총 자본의 **3%** 초과 손실 시 당일 매매 즉시 중단.

| 매도 수수료 | 0.015% | 0.015% | |
| 증권거래세 | 0.18% | 없음 (일부) | 2025년 기준 |
